## sideChannel

Simple demo of side channel attack in python

### Input Keystrokes :

	q X 10 times
	c X 10 times
	y X 10 times
	m X 10 times
	p X 10 times
	5 X 10 times
	{q,c,y,m,p,5} X 4 times

### Audio file specification

	Num of channels    :  1
	Sample Width       :  2
	Sampling Frequency :  44100
	Number of frames   :  7185023
	Compression type   :  NONE
	compression name   :  not compressed
